# Echo Bot — кратко

Мини-бот на Python с aiogram 3.  

**Функции:**
- `/start` — приветствие и начало регистрации (FSM)
- `/help` — справка
- Регистрация пользователя: имя и номер телефона через кнопку
- Фото: бот показывает `file_id` и пересылает фото
- Меню с кнопками и inline кнопками (Nike, Adidas, Reebok)
- Эхо: все остальные сообщения бот повторяет пользователю

**Технологии:** Python, aiogram, python-dotenv, FSM, клавиатуры

**Запуск:**
1. Установить зависимости: `pip install aiogram python-dotenv`
2. Создать `.env` с `BOT_TOKEN`
3. Запустить: `python echo_bot.py`
