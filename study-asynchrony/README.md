# Асинхронность в Python — кратко


## Синхронный код

Синхронный код — это код, который выполняется **пошагово**, строка за строкой.  
Каждая операция **блокирует** выполнение следующей до своего завершения.  

Пример:

```python
print("Начало")
x = 10 / 2    # здесь программа ждёт окончания деления
print("Конец")
```
